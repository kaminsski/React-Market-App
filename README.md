# React-Market-App
 React-Market-App website is built with the MERN stack

## Functions/features of the website
* Login/Register
* Cart
* Admin-(add/delete/update product)
* Profile


## Technologies used


* HTML5
* CSS3
* JavaScript (ES6)
* GitHub
* React
* Tailwind
* MongoDB
* NodeJS
* ExpressJS
* Bcrypt
* JWT
* Validator
